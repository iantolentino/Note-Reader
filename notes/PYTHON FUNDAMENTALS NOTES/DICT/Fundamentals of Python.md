## Syntax and Errors

### Comments 
- are completely ignored. They're entirely for the use of programmers like you
- can be written using the hash character(#)
- are not read by the interpreter, means you can write whatever you want and the computer will ignore it.
### Indentation
- in python indentation is extremely important because we use them to create new block of code.
- required for indicating a block of code
### Parentheses 
- in the print() function indicate that we need to put something within them.
### Error
- results form python rules not being followed in our code.
### Block (of code)
- refers to code that belongs to the same line, and are indented
