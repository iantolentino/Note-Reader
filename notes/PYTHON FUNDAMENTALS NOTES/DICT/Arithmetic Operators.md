- are operators that are used to perform mathematical operations

### Addition Operator
- an operator that outputs sum of values
- used for concatenation

### Subtraction Operator
- an operator that outputs the difference of values

### Multiplication Operator
- is an operator that outputs the product of values

### Division Operator
- an operator that outputs the quotient of values

### Modulus
- an operator that outputs the remainder of a quotient

### Exponentiation Operator
- an operator that allows you to use the second value as an exponent

### Floor Division
- an operator that outputs the rounded-off value

### Precedence
- the order by which operations are performed
