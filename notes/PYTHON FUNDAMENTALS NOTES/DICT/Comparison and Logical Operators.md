### Comparison Operators/Relational Operators
-  compare two values and indicate the difference between the two values
- they return either true or false

#### Equal Operator ==
- indicates that two values are the same
#### Not Equal Operator !=
- this indicates that two values are not equal to each other
#### Greater than Operator >
- determines if the first values is larger than the other
#### Less than Operator <
- determines if the first values is smaller than the other
#### Greater than or equal to Operator >=
- determines if the first values is greater than or equal to the other value
#### Less than or equal to Operator <=
- determines if the first values is less than or equal to the other value


### Logical Operators
- combine conditional statements

#### And Operator and
- checks if both statements are true
- used to ensure that, when you check multiple conditions, both sides of the conditions are true
#### Or Operator or
- checks if only one of the statements is true
- if at least one condition is true, the code will run, but if both condition are false the code will not run
#### Not Operator not
- reverse the result
