### IF keyword 
- used for decision making operators
- code would run only if the condition is met
- works the same way that a decision is made
- a condition will be checked. If that condition is true, the task will be performed, otherwise, you can move on without performing the task

NOTE
- indentation is very important.
- the line after the if statement must always be indented or else the code will not run
### Else keyword
- provides an alternative solution in case both of the if and elif are false
- the end all be all of the if statement [....]
- covers all other possibilities not covered and will always run the code if the program gets to it
### Elif keyword
- short for else if, this gives an alternate condition so that the code can have another chance to run
- ability to run separate block of code depending on the condition. They are also known as else is statements
