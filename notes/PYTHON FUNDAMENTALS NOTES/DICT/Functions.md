### Parameters 
- is a piece of information the function needs to do its job
- this information is contained within the parentheses of the function

### Def Keyword
- allows user to define functions
- def keyword is used to define a function. It allows you to create your own function
- allows us to take a long block of code and shorten it so it can be used again if needed
### Arguments
- a piece of information that's passed from a function call to a function
- this supplies the information the parameter requires. For example, if the parameter is "name, " the argument would be "Arnold"

note: if your function does not get the argument it expects, ruining your code will lead to an error
